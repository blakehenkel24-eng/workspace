```
Background:      #0D1117  (GitHub dark bg)

Border Active:   #58A6FF  (Focus rings)

Text Primary:    #C9D1D9  (Headings, body)
Text Secondary:  #8B949E  (Labels, meta)
Text Muted:      #6E7681  (Timestamps, hints)

Accent Blue:     #58A6FF  (Primary actions)
Accent Green:    #238636  (Success, active)
Accent Amber:    #D29922  (Warnings, blocked)
Accent Red:      #DA3633  (Errors, offline)

### Typography

```
Font Stack:      -apple-system, BlinkMacSystemFont, "Segoe UI", Helvetica, Arial, sans-serif

  S:  12px / Meta, timestamps
  XS: 11px / Badges, labels

### Spacing Scale

```
4px   - Tight (icon padding)
8px   - Compact (list items)
12px  - Default (card padding)
16px  - Comfortable (section gaps)
24px  - Spacious (major sections)
32px  - Layout (page margins)
```

### Border Radius

```
Small:   4px  (Buttons, inputs)
Medium:  6px  (Cards, panels)
Large:   8px  (Modals, toasts)
Full:    50%  (Avatars, status dots)
```

## 7. Responsive Considerations

### Breakpoints

| Breakpoint | Layout Changes |
|------------|----------------|
| `>1400px` | Full layout, all columns visible |
| `1200px` | Kanban: horizontal scroll columns |
| `992px` | Sidebar collapses to icons |
| `768px` | Mobile: stack layouts, bottom nav |
| `<480px` | Minimal: single column, essential only |

### Mobile Adaptations

- Dashboard: Cards stack vertically
- Kanban: Single column view with column switcher
- Chat: Full-screen threads, back button to list
- Agent detail: Tabbed interface, one section at a time
