✅ **DO:**

✅ **DO:**

- Keep specific and actionable

## 6. URL STRUCTURE RECOMMENDATIONS

### URL Structure Principles

**Formula:** `slidetheory.io/[category]/[page-name]`

### Recommended URL Patterns

| Page Type | URL Pattern | Example |
|-----------|-------------|---------|
| Homepage | / | slidetheory.io |
| Product Features | /features | slidetheory.io/features |
| Specific Feature | /features/[feature-name] | slidetheory.io/features/ai-generator |
| Use Case | /for/[audience] | slidetheory.io/for/private-equity |
| Framework Guide | /frameworks/[framework-name] | slidetheory.io/frameworks/mece-principle |
| Blog Post | /blog/[post-slug] | slidetheory.io/blog/mece-principle-guide |
| Template Category | /templates/[category] | slidetheory.io/templates/mckinsey-style |
| Pricing | /pricing | slidetheory.io/pricing |

### URL Best Practices

✅ **DO:**
- Use lowercase letters
- Use hyphens (-) not underscores (_)
- Keep URLs under 75 characters
- Include target keyword
- Use HTTPS

❌ **DON'T:**
- Use spaces or special characters
- Include unnecessary parameters
- Create deep nesting (max 3 levels)
- Use dates in URLs (unless news)
- Change URLs after publishing (without redirects)

### URL Examples

| Page | ✅ Good URL | ❌ Bad URL |
|------|-------------|------------|
| MECE Guide | /frameworks/mece-principle | /frameworks/mece_principle_guide_v2 |
| AI Generator | /features/ai-slide-generator | /features/product?id=123&type=ai |
| PE Landing Page | /for/private-equity | /for/privateequitypresentations2026 |
| Blog Post | /blog/consulting-slide-tips | /blog/2026/02/04/post-12345 |

## 8. CONTENT OPTIMIZATION CHECKLIST

### Keyword Placement

For each page, ensure:

- [ ] Primary keyword in H1 (within first 60 chars)
- [ ] Primary keyword in first 100 words
- [ ] Primary keyword in at least one H2
- [ ] Primary keyword in conclusion/CTA section
- [ ] Keyword density: 1-2% (natural usage)
- [ ] LSI/related keywords throughout content

### Content Quality Standards

✅ **MUST HAVE:**
- Minimum 800 words for blog posts (1500+ for guides)
- Unique, original content (no duplication)
- Clear structure with headers
- Bulleted/numbered lists for scannability
- Visual elements (images, diagrams) every 300 words
- Internal links (minimum 3-5 per page)
- External links to authoritative sources (2-3 per page)
- Clear CTA at the end

### Content Freshness

- [ ] Review and update top 10 pages quarterly
- [ ] Update statistics and examples annually
- [ ] Add "Last Updated" date to evergreen content
- [ ] Refresh blog posts that drop in rankings
