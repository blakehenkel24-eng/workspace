# Memory

This folder contains daily session logs and memory files.

## Files

| File | Purpose |
|------|---------|
| `MEMORY.md` | Curated long-term memory (at root) |
| `SEMANTIC-MEMORY.md` | Semantic memory configuration |
| `YYYY-MM-DD.md` | Daily session logs |

## Structure

```
memory/
├── SEMANTIC-MEMORY.md      # Semantic memory config
├── 2026-02-04.md          # Example daily log
├── 2026-02-05.md          # Example daily log
└── ...                     # More daily logs
```

## Usage

- **Daily logs**: Created automatically each session
- **Long-term memory**: See `../MEMORY.md` at repository root
- **Semantic search**: Configured in `SEMANTIC-MEMORY.md`
