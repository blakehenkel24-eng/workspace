# Workspace

Personal workspace for AI-assisted development, product management, and knowledge organization.

## Structure

```
workspace/
├── README.md              # This file
├── .gitignore             # Git ignore rules
├── docs/                  # Documentation & guides
│   ├── AGENTS.md          # Agent workspace guide
│   ├── SOUL.md            # Agent identity/personality
│   ├── USER.md            # User context
│   ├── TOOLS.md           # Tool notes & environment specifics
│   ├── HEARTBEAT.md       # Periodic check reminders
│   ├── IDENTITY.md        # Identity configuration
│   ├── MEMORY.md          # Long-term memory (in memory/ folder)
│   ├── guides/            # MASTER_GUIDE content
│   └── to-implement/      # Documentation to implement
├── memory/                # Daily notes & logs
│   ├── MEMORY.md          # Curated long-term memory
│   └── YYYY-MM-DD.md      # Daily session logs
├── products/              # Product work
│   └── slidetheory/       # SlideTheory product
│       ├── docs/          # Product specs
│       ├── marketing/     # Marketing materials
│       ├── mvp/           # MVP implementation
│       ├── research/      # Research & routing docs
│       ├── archive/       # Archived files
│       ├── general/       # General assets
│       ├── ideas/         # Ideas & concepts
│       ├── index.html     # Main dashboard
│       ├── README.md      # Product readme
│       └── TODO.md        # Product todo list
├── tools/                 # Tools & utilities
│   └── mission-control/   # Dashboard interface
├── archive/               # Archived content
├── skills/                # Skill definitions
├── .clawhub/              # Clawhub data
└── .learnings/            # Learning records
```

## Quick Reference

| Location | Purpose |
|----------|---------|
| `docs/` | System documentation, guides, and reference |
| `memory/` | Daily logs and curated memories |
| `products/` | Active product work (SlideTheory, etc.) |
| `tools/` | Utilities and tools |
| `archive/` | Old/stale content |
| `skills/` | Reusable skill definitions |

## Usage

1. **Start here**: `docs/AGENTS.md` for agent onboarding
2. **Daily work**: Check `memory/` for recent context
3. **Product work**: Navigate to `products/slidetheory/`
4. **Tools**: Access `tools/mission-control/` for dashboards
